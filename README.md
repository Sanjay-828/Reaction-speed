# ⚡ Reaction Time Test

A simple web game that measures how fast you react when the screen turns green.

## 🎮 How to Play

1. Click **Start**
2. Wait for the screen to turn **green**
3. Click as fast as possible
4. Your **reaction time (ms)** will be displayed

## ✨ Features

* Random delay before the signal
* Reaction time measurement in milliseconds
* Best score saved in browser
* Top 5 leaderboard
* Responsive design
